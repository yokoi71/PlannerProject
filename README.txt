How to run the college planner software. (Frontend: Angular, Backend: Spring Boot and Spring Security, Database: MySQL)


1 Start database.

Start MySQL Command Line Client (or equivalent) and enter username/password (pwd: yokorootpass!123)
Select college_planner to use. ("USE college_planner")

2. Start the backend application.
Open Eclipse and import CollegePlannerBackend (C:\Yoko\Programming\Projects\PlannerProject\CollegePlannerBackend).
Start the application (highlight CollegePlannerBackendApplication.java in the package workplace window, right-click to execute). 

3. Start the frontend application.
Open Visual Studio Code, then go [File]->[Open Folder] to open the " C:\Yoko\Programming\Projects\PlannerProject\CollegePlannerFrontend" folder.
Open a terminal and "npm install" and "npm start"

4. Open the browser to use.
On the browser, type "http://localhost:4200" to use the software.
